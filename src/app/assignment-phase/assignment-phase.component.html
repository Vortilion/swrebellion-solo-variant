Assignment Phase

Assign Leaders to Missions Rebel player assigns leaders to missions Create Mission Deck Draw a specified number of Missions (see Missions Table, below) shuffle the drawn cards, and place them in a facedown pile, forming the Initial Missions Deck Mission Table Turn 1: 2 Missions (1 Starting Mission + 1 Main Mission) Turns 2-3: 3 Missions (1 Starting Mission + 2 Main Missions) Turn 4+: 4 Missions (1 Starting Mission + 3 Main Missions) Rebel Base Revealed 3 Missions (1 Starting Mission + 2 Main Missions) Play Action Cards The Empire plays all Action Cards designated as playable during the Assignment Phase
